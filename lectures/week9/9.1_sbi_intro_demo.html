Simulation-Based Inference (SBI) with variational inference¶

We'll go through some examples of Bayesian inference with SBI using variational inference to estimate the joint $p(\theta,\mathcal{D})$. We'll use the same examples as in the ABC demo and use pytorch.

Example 0¶

Let's return to the example from a few weeks ago

Imagine a hypothetical simplistic example, where we point our telescope to observe the light from a single star with true flux $F_{\rm true}$, which remains constant with time. We will ignore any systematic errors. We perform $N$ measurements with our telescope, where the $i^{\rm th}$ measurement report an observed flux $F_i$ with uncertainty/error $e_i$.

Lets generate a synthetic observation

For our prior, lets use a simple Gaussian $$p(\theta) = \mathcal{N}(1002, 4)$$ This would hypothetically be measurement from a prior experiment

Then by repeating the following two steps,

  1. sample the prior: $\theta'\sim p(\theta)$
  2. run simulation: $\mathcal{D}' \sim F(\theta')$

We can sample $(\theta', \mathcal{D}') \sim p(\theta, \mathcal{D})$

variational inference¶

Now lets use variational inference to estimate this joint distribution.

First, we need to select our surrogate distribution $q_\phi(\theta, \mathcal{D})$.

We'll use a simple 2D Gaussian, so the hyperparameter $\phi$ is the mean and covariance matrix

In [8]:
def log_q(x, mean, cov):
    # log q_phi
    diff = x - mean               

    # cholseky factorization to lower-triangular matrix 
    L = torch.linalg.cholesky(cov) 
    
    # triangular solve trick so we don't 
    # have to invert the covariance matrix 
    sol = torch.linalg.solve_triangular(
        L, diff.T, upper=False).T

    # log det --- det(Cov) = det(L)^2
    logdet = 2.0 * torch.sum(torch.log(torch.diag(L)))

    D = mean.shape[-1]
    return -0.5 * (D * torch.log(torch.tensor(2 * torch.pi)) + logdet
                   + torch.sum(sol ** 2, dim=-1))

Now lets determine $\phi$, the mean and covariance, by minimizing the KL divergence between $p(\theta, \mathcal{D})$ and $q_\phi(\theta, \mathcal{D})$:

$$D_{\rm KL} (p\parallel q_\phi) \approx \sum\limits_{(\theta',\mathcal{D}')\sim p(\theta, \mathcal{D})} \left[ - \log q_\phi(\theta',\mathcal{D}')\right]$$

Again, note that we're estimating the KL divergence using $(\theta',\mathcal{D}')\sim p(\theta, \mathcal{D})$ --- i.e. simulation-based inference!

In [9]:
def KL(x, mu, A):
    '''KL divergence'''
    # some more common linear algebra tricks
    # A @ A.T ensure cov is positive semi-definite
    # 1e-3 * I ensures its symmetric positive definite
    cov = A @ A.T + 1e-3 * torch.eye(A.shape[0], device=A.device)
    return -torch.mean(log_q(x, mu, cov))

Now you can see why I chose Gaussian priors earlier.

With uniform priors, we're confronted with a key challenge with variational inference.

Our multivariate Gaussian surrogate cannot estimate $p(\theta, \mathcal{D})$ accurately. In this case, there's no way to accurately estimate the posterior with $q_\phi$.

Since $q_\phi$ does not accurately estimate $p(\theta, \mathcal{D})$, we cannot proceed with the Bayesian inference

unit summary¶

  • SBI enables us to sample $p(\theta, \mathcal{D})$ even when we don't have access to the explicit likelihood
  • Using variational inference we can estimate $p(\theta, \mathcal{D})$ with a surrogate $q_\phi$ that is easy to sample, then use ABC for Bayesian inference
  • we can implement SBI with VI in pytorch
  • caution: SBI + VI, as with VI, is sensitive to the choice of $q_\phi$
