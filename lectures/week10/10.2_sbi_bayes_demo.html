sbi¶

We'll explore SBI techniques such as neural posterior estimation and neural likelihood estimation using the sbi python package: https://sbi.readthedocs.io/en/latest/. We'll go through the sample inference examples as the SBI with variational inference demo.

Example 0¶

Let's return to the example from a few weeks ago

We point our telescope to observe the light from a single star with true flux $F_{\rm true}$, which remains constant with time. We will ignore any systematic errors. We perform $N$ measurements with our telescope, where the $i^{\rm th}$ measurement report an observed flux $F_i$ with uncertainty/error $e_i$.

Lets generate a synthetic observation

In [5]:
F_true = 1000
F_obs = np.random.poisson(lam=F_true)
In [6]:
# simulator of the data generation process
def simulation(F_true): 
    return np.random.poisson(lam=F_true)

For our prior, lets use a simple Gaussian $$p(\theta) = \mathcal{N}(1002, 4)$$ This would hypothetically be measurement from a prior experiment

In [7]:
def sample_prior(): 
    # sample prior
    return 1002. + 4. * np.random.normal()

Let's sample $p(\theta, \mathcal{D})$

In [8]:

samples = np.zeros((Nsample, 2))

samples = torch.tensor(samples, dtype=torch.float32)

Now what happens if we change the prior to the uniform priors. Recall that this was problematic for SBI with VI due to the limited flexibility of our surrogate

def simulation(theta): 
    a, b = theta
    noise = yerr * np.random.normal(size=len(x))
    return a + b * x + noise

def sample_prior(): 
    # sample a and b from uniform prior
    _a = -0.5 + np.random.uniform()

    return [_a, _b]
